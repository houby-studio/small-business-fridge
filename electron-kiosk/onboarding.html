Small Business Fridge — Kiosk

Initial Setup

Configure the kiosk before first use. Settings can be changed later with snap set sbf-kiosk key=value.

Connection

The address of the Small Business Fridge web server this kiosk will load.

Extra origins the kiosk may navigate to (comma-separated). Leave blank if not needed.

► Advanced settings

Display & Language

Language code passed to the browser (e.g. cs, en, de).

Audio

ALSA device string. auto uses the system default. Check available devices with aplay -l. Examples: hw:0,0 (analog), hw:0,3 (HDMI).

Volume level applied at startup (0–100).

Service

Start automatically on boot (daemon mode). Disable to launch manually.
